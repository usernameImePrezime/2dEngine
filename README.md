# 2dEngine

Neki moj pokušaj da napravim 2D engine u C#. Mučio sam se sa sudarima pa sam ubacio QuadTree jer je bez toga sve kočilo čim ubacim više od 50 loptica.

Some of my attempt to make a 2D engine in C#. I was struggling with collisions so I put in a QuadTree because without it everything would brake as soon as I put in more than 50 balls.
